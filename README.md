# amcharts5-skill
Claude AI Skill for building amCharts 5 code

# Installing the amcharts5 skill
1. Download the skill:
 - Clone repo: `git clone https: /github.com/amcharts/amcharts5-skill`
 - Or download ZIP from Releases
2. Install in Claude:
 - Open Claude.ai > Settings > skills
 - Click "Upload skill"
 - Select the skill folder (zipped)
 3. Enable the skill:
  - Toggle on the amcharts5-skill skill
  - Ensure your MCP server is connected
 4. Test:
  - Ask Claude: "Set up a new project in amcharts5-skill"
